Class: DishDetails

DishDetails()

new DishDetails()

This component is nested in TransactionsDishItem and ListingsDishItem. It displays dish details in a pop-up dialog. Allows users to contact seller and purchase the dish.

Props

Name Type Default value Required? Description
name String - Yes Name of the dish
description String - Yes Description of the dish
locationLat Number - Yes Location (latitude) of the dish
locationLong Number - Yes Location (longitude) of the dish
price Number - Yes Price of the dish
dietaryRestrictions Array.<String> - Yes List of dietary restrictions on the dish
ingredients Array.<String> - Yes List of ingredients in the dish
url String - Yes Image URL of the dish
maxQuantity Number - Yes Total number of dishes that can be purchased
sellerId String - Yes Dish seller ID
sellerPaypalId String - Yes Dish seller PayPal ID
id String - Yes Dish ID

Data

Name Type Default value Description
dishDetailsDialog Boolean - Designates whether dish details dialog is open or not
quantity Number - Number of dishes to be purchased

Computed

Name Type Description
chatId String Corresponding chatId for communicating with the seller
