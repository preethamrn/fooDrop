Module: ChatBase

It is located at /chat/:id. Shows all the chat messages between a buyer and seller.

Data

Name Type Default value Description
chatId String - Identifier for receiving chat messages from the backend
message String - Current message stored in the text field on the page
chats Array.<Chat> - List of chats containing the chat message, userId, and username of the user sending the message
buyer String - Buyer name in the conversation
seller String - Seller name in the conversation
socket Socket.io - Socket connection object for real time communication with the server
