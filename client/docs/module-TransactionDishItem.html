Module: TransactionDishItem

This component is nested within the UserProfileBase component. It displays a single row of details for a transaction in the transactions list.

Props

Name Type Default value Required? Description
name String - Yes Name of the dish
location Position - Yes Location of the dish in terms of { latitude, longitude }
dish Dish - Yes Dish details of the transaction

Data

Name Type Default value Description
distanceInMiles Number - Distance of the dish from user location

Computed

Name Type Description
cost Number Cost of the transaction. Negative if the dish is bought and positive otherwise.
