Module: ListingsDishItem

This component is nested within the ListingsBase component. It displays a single row of details for a dish in the dishes list.

Props

Name Type Default value Required? Description
name String - Yes Name of the dish
location Position - Yes Location of the dish in terms of { latitude, longitude }
price Number - Yes Price of the dish
dish Dish - Yes Dish details of the dish

Data

Name Type Default value Description
distanceInMiles Number - Distance of the dish from user location

Methods

openDishDetails()

Opens the Dish Details dialog. Used by GoogleMapsMarker to expose an API to the DishDetails.
