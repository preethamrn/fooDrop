Class: SearchDishBase

SearchDishBase()

new SearchDishBase()

Component for searching for dishes. Allows a user to enter search parameters which will redirect them to ListingsBase with their appropriate search results.

Data

Name Type Default value Description
valid Boolean - Validates the search details input form
searchDishName String - Name of the dish in the search response
searchDishLocationLat Number - Location (latitude) of the search request
searchDishLocationLong Number - Location (longitude) of the search request
searchDishIngredients Array.<String> - Ingredients to include in the search response
searchDishDietaryRestrictions Array.<String> - Dietary restrictions required in the search response
searchDishPrice Array.<Number> - Price range of the search request
searchDishRadius Number - Radius of the search request

Computed

Name Type Description
storeUserState String Watcher function that checks for modifications to Vuex state
